## ✨ Eliminate Travel Decision Fatigue with AI

GoNiva is an AI-powered Online Travel Agency (OTA) that helps travelers overcome decision fatigue and information overload by providing hyper-personalized travel recommendations through natural conversations. We present results in a carousel format (5 options at a time) for easier decision-making.

## 🎯 Key Features

*   **AI-Driven Personalization:** Understands your travel preferences through natural language.
*   **Carousel Interface:** Presents a curated selection of 5 options to simplify choices.
*   **Reduces Decision Fatigue:**  AI curates the best options, saving you time and stress.

## 💡 Problem

*   76% of global travelers want apps that reduce travel friction ([Hilton](https://stories.hilton.com/emea/releases/2024-trends-report)).
*   Travel planning has become overwhelming for consumers ([TravelPulse](https://www.travelpulse.com/news/features/travel-planning-has-become-overwhelming-report-shows)).
*   52% abandon online bookings due to bad experiences ([Siteminder](https://arc.net/l/quote/uayjrcki)).

## 🚀 Getting Started

### Installation (Example - Adapt to your actual setup)

1.  Clone the repository:

    ```bash
    git clone https://gitlab.com/goniva/goniva.git
    cd goniva
    ```

2.  Install dependencies (example using `npm`):

    ```bash
    npm install  # Or, if using Yarn: yarn install
    ```

3.  Set up your environment variables (API keys, etc.).  See `.env.example` for a template.

4.  Run the application:

    ```bash
    npm start  # Or, yarn start
    ```

### Usage

1.  Start a conversation with GoNiva's AI.
2.  Describe your trip (dates, budget, interests, events).
3.  Niva(AI Assistant) will have a dynamic conversation with the user to understand their needs. 
4.  GoNiva will present 5 personalized options in a carousel.
5.  Refine your search through by saving or asking for new carousels.
6.  Book your perfect trip!
